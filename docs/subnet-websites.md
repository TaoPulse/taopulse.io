- **SN001 (Apex):** The original SN1 "text prompting" subnet. Has no dedicated project website — taostats.io is the best reference.
- **SN011 vs SN051:** Both reference Celium/lium. SN11 is Dippy (AI companion) built on Celium; SN51 is Celium itself.
- **SN013:** Macrocosmos / Data Universe — API we evaluated for social feed integration (see backlog TP-015).
- **SN081:** Patrol is built by TAO.com team. tao.com is the mobile wallet app — likely the right association but needs confirmation.
